# Running Cloud Foundry
[BOSH](https://docs.cloudfoundry.org/bosh/) is the recommended and supported means to [deploy Cloud Foundry](https://docs.cloudfoundry.org/deploying/). However, BOSH is more than a provisioning and deployment tool. It is also the tool you use to manage and troubleshoot existing Cloud Foundry instances.
